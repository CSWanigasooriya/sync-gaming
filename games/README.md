## Example: Minimal WebGL Game

You can modify:

Create `games/hello-world/index.html`:- Colors in the `colors` array (RGB values 0-1)

- Rotation speed in `render()` function (change `angle += 0.01`)

```html- Cube size by scaling vertices

<!DOCTYPE html>- Camera position in matrix calculations

<html>

<head>## License

    <meta charset="UTF-8">

    <title>Hello World Game</title>MIT - Feel free to use and modify!

    <style>
        body { margin: 0; }
        canvas { display: block; width: 100%; height: 100vh; }
    </style>
</head>
<body>
    <canvas id="canvas"></canvas>
    <script>
        const canvas = document.getElementById('canvas');
        const gl = canvas.getContext('webgl');
        
        // Your WebGL code here
        gl.clearColor(0.0, 0.0, 0.0, 1.0);
        gl.clear(gl.COLOR_BUFFER_BIT);
    </script>
</body>
</html>
```

Then package: `npm run package-game hello-world`

## Workflow for Developers

1. Create your game in `/games/your-game/`
2. Test it locally: Open `games/your-game/index.html` in your browser
3. Package it: `npm run package-game your-game`
4. Upload the `.zip` file through the Admin Panel
5. Share the game link!

## Tips

- Keep file sizes small for better performance
- Use compressed images and assets
- Test in multiple browsers before uploading
- Avoid external APIs that require CORS configuration
- For multiplayer, consider using WebSockets
